## 🚀 Comandos de Control

Para que el sistema funcione al 100%, ejecutamos estos dos motores en la terminal:

### 🌐 1. Visualización Web
```bash
npx serve public
```
* **¿Para qué sirve?**: Levanta un servidor web local que proyecta tu carpeta `public`.
* **Función**: Permite que el archivo `index.html` se convierta en una página funcional accesible desde el navegador, mostrando los gráficos neón y el estado de cuenta.

### 🤖 2. Motor del Bot
```bash
node bot.js
```
* **¿Para qué sirve?**: Activa la lógica de programación en el servidor (Node.js).
* **Función**: Conecta tu código con la API de Telegram y Firebase. Es el encargado de enviar alertas, procesar comandos y mantener la seguridad de los datos en tiempo real.
